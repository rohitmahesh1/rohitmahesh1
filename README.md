# Hi, I'm Rohit

I build software at the intersection of machine learning, data-driven tools, and user-centered design.

My projects often focus on turning complex technical workflows into systems that are easier to use, inspect, and learn from — whether that means analyzing biological wave data, building simulation platforms, or connecting vision models to mobile applications.

This GitHub contains a mix of research code, product-oriented experiments, and tools developed from genuine technical curiosity.

## Selected Projects

### [WaveCalling](https://github.com/rohitmahesh1/WaveCalling)
A pipeline for converting raw kymographs into structured wave statistics and visual outputs.

I rebuilt the workflow in Python and ONNX and paired it with a lightweight web interface, making the process more usable for analysis and review.

### [Epidemic Lab](https://github.com/rohitmahesh1/epidemic_lab)
An interactive SEIR / SEIRV simulation platform with a React frontend and a Python backend deployed on Cloudflare Workers.

This project focused not only on the underlying model, but also on the surrounding product experience: session state, scenario comparison, charts, and report workflows.

### [Eco Health](https://github.com/rohitmahesh1/ecobuddy-ios)
A SwiftUI iOS application centered on eco-health education, guided challenges, pledges, and awareness content.

### [InternVL Scene Describer](https://github.com/rohitmahesh1/InternVL-SceneDescriber)
A scene-description and visual question answering server built on top of InternVL, designed to work with a companion iPhone app over gRPC.

The goal was to connect model inference to a practical interface that could return useful language from real-world images.

### [Mouse/BCI Trajectory OCR](https://github.com/rohitmahesh1/neuralink_solution)
A trajectory-to-text OCR pipeline that converts movement data into rendered word images and decodes them with TrOCR.

This project sits at an interesting intersection of signal processing, modeling, and applied engineering.
